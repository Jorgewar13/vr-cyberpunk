Ciudad Cyberpunk
Este proyecto consiste en una experiencia de realidad virtual (VR) que te sumerge en una ciudad futurista de estilo cyberpunk. Utiliza la tecnología A-Frame para crear un entorno inmersivo.

Requisitos
Navegador web compatible con WebVR.
Conexión a internet para cargar recursos externos.
Instalación
Clona este repositorio en tu máquina local:
bash
Copy code
git clone https://github.com/tu_usuario/ciudad-cyberpunk.git
Abre el archivo index.html en tu navegador web compatible con WebVR.
Uso
Una vez que el proyecto esté cargado en tu navegador, podrás explorar la ciudad cyberpunk utilizando los controles de tu ratón o teclado.
Interactúa con los elementos en la escena haciendo clic en ellos para activar sonidos o mensajes.
Para moverte, utiliza las teclas W, A, S y D o los controles de tu ratón.
Contribución
¡Las contribuciones son bienvenidas! Si deseas mejorar este proyecto, sigue estos pasos:

Haz un fork del repositorio (https://github.com/tu_usuario/ciudad-cyberpunk/fork)
Crea una rama para tu nueva funcionalidad (git checkout -b feature/nueva-funcionalidad)
Realiza los cambios necesarios y haz commit de tus cambios (git commit -am 'Agrega nueva funcionalidad')
Haz push a la rama (git push origin feature/nueva-funcionalidad)
Crea un nuevo Pull Request
